The technology

Craftables is my first major project. The web app allows users to upload/edit a project, leave comments, and upload a bio. Nonusers can view projects and leave comments. To complete this project, I use the full extent of my knowledge from App Academy. The back end was built using Ruby on Rails and PostgreSQL database. I used React framework to build a responsive single page app.
